# Pasha

Windows 11 で動かなくなった定番キャプチャソフト **WinShot** にインスパイアされた、軽量なスクリーンキャプチャ常駐ツールです。タスクトレイに常駐し、グローバルホットキーとウィンドウ操作の両方でキャプチャできます。

> 本ソフトは WinShot へのリスペクトに基づく**独自実装**であり、オリジナルの WinShot とは無関係です。

---

## 特長

- **4 種類のキャプチャ対象**
  - デスクトップ全体（マルチモニタを 1 枚にまとめて取得）
  - アクティブウィンドウ（DWM で影を除いた実表示範囲）
  - クライアント領域（タイトルバー・枠を除いた中身だけ）
  - 矩形範囲指定（ドラッグで選択）
  - さらに **モニター個別**の指定キャプチャにも対応
- **出力先**: ファイル保存 / クリップボードへコピー（同時コピーも可）
- **保存形式**: PNG / JPEG（JPEG は画質指定可）
- **連番ファイル名**（接頭語 + 番号、桁数指定）または **日時ファイル名**
- **遅延キャプチャ**（0 / 3 / 5 / 10 秒）
- **グローバルホットキー**を機能ごとに自由割り当て（WinShot のホットキー設定を踏襲）
- **高 DPI / マルチモニタ対応**（実解像度で鮮明に取得）
- **タブ付きの操作ウィンドウ**で全機能を操作
- 設定は `%AppData%\Pasha\config.ini` に保存
- .NET Framework 4.8 同梱の Windows 11 で **追加インストール不要**で動作

---

## 動作環境

- Windows 10 / 11（.NET Framework 4.8 同梱）

---

## ダウンロード / インストール

[Releases](https://github.com/peresimmon/pasha-screenshot/releases) から `Pasha.exe`（と `Pasha.exe.config`）をダウンロードし、任意のフォルダーに置いて実行するだけです。インストール作業は不要です。

> `Pasha.exe.config` は高 DPI 表示の最適化に使われます。同じフォルダーに置いてください（無くても動作します）。

## 使い方

起動するとタスクトレイにカメラ型アイコンが常駐します。アイコンを**ダブルクリック**すると操作ウィンドウが開きます。

### 既定のホットキー（「ホットキー」タブで変更可）

| 機能 | 保存 | クリップボード |
|------|------|----------------|
| デスクトップ全体 | `Ctrl + Shift + 1` | `Ctrl + Alt + 1` |
| アクティブウィンドウ | `Ctrl + Shift + 2` | `Ctrl + Alt + 2` |
| 矩形範囲指定 | `Ctrl + Shift + 3` | `Ctrl + Alt + 3` |
| クライアント領域 | `Ctrl + Shift + 4` | `Ctrl + Alt + 4` |

- ホットキー欄をクリックして割り当てたいキーを押すと設定できます（`Delete` で解除）。
- `PrintScreen` / `Alt + PrintScreen` などの割り当てにも対応しています。

### ウィンドウからの操作

- **キャプチャ**タブ: 出力先・遅延・モニターを選んでボタンで撮影
- **基本設定**タブ: 保存先フォルダー、接頭語、ファイル名形式（連番／日時）、連番桁数、保存形式、JPEG 画質
- **ホットキー**タブ: 機能ごとのホットキー割り当て
- **動作設定**タブ: 起動時トレイ格納、×でトレイ格納、保存時クリップボード同時コピー、保存後フォルダーを開く、シャッター音、通知
- **情報**タブ: バージョン情報・保存先を開く

## ソースからのビルド

### 方法 1: build.bat（SDK 不要・推奨）

Windows 標準の .NET Framework 付属コンパイラ（`csc.exe`）でそのままビルドできます。

```bat
build.bat
```

`Pasha.exe` と `Pasha.exe.config` が生成されます。

### 方法 2: .NET SDK / Visual Studio

```bat
dotnet build -c Release
```

## プロジェクト構成

```
Pasha/
├─ src/
│  ├─ Program.cs          … エントリポイント / DPI 設定 / 多重起動防止
│  ├─ AppController.cs     … トレイ常駐・ホットキー・キャプチャ制御
│  ├─ MainForm.cs          … タブ付き操作ウィンドウ
│  ├─ CaptureEngine.cs     … 画面取得 (デスクトップ/ウィンドウ/クライアント)
│  ├─ RegionSelectForm.cs  … 矩形範囲選択オーバーレイ
│  ├─ HotkeyWindow.cs      … グローバルホットキー受信
│  ├─ HotkeyBox.cs         … ホットキー割り当て用コントロール
│  ├─ Config.cs            … 設定の読み書き・ホットキー定義
│  └─ Util.cs              … アイコン生成・アプリ情報
├─ app.manifest            … DPI / 実行レベル
├─ app.config             … 高 DPI 自動リサイズ設定
├─ build.bat              … csc によるビルド
├─ Pasha.csproj        … SDK 向けプロジェクト
└─ .github/workflows/     … CI (ビルド・リリース)
```
